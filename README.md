# sciencebowl-buzzer
A buzzer system to practice for national science bowl. 


This works off an esp32s, connected through websockets to a nodejs server. 

index.html is the frontend, generated by claude (the goat) 
  in order to make it work, go to line 519 and change the IP to your local IP

main.cpp is the code to run on your esp32
  replace lines 4, 5, 6, and 8 to make it run
